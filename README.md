# Multi-Focus Image Fusion Dataset

## Dataset Information

This dataset contains **150 images** specifically designed for Multi-Focus Image Fusion algorithms.

**Key Feature:** Unlike other datasets in this field, this dataset includes **more than two images** for fusion, making it unique and valuable for advanced multi-focus image fusion research.

**Dataset Prepared by:** Samet Aymaz
